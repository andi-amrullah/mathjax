F4 Solving Solving Quadratic Equation Introduction



Learning objectives

This unit is focused on methods of solving quadratic equations. After completing Unit F4 you should

  • be able to solve quadratic equations that factorise into two linear terms

  • be able to use the formula for solving quadratic equations

  • understand how to complete the square for a quadratic and hence solve the equation.

Introduction

We can tell from Old Babylonian clay tablets dating from around 2000 BC that the Babylonians knew how to solve a pair of simultaneous equations of the form

\( \hspace{11cm} x + y = p, xy = q \)
which is equivalent to the equation
\( \hspace{11cm} x^2 + q = px \)

In the 8th Century BC, quadratic equations of the form \( ax^2 = c \) and \( ax^2 + bx = c \) were explored in ancient India, using geometric methods. Babylonian mathematicians from circa 400 BC and Chinese mathematicians from circa 200 BC used the method of completing the square (see Section F4.3) to solve quadratic equations with positive roots, but did not have a general formula.

In 628 AD, Brahmagupta, an Indian mathematician, gave the first explicit solution of the quadratic equation but it was not until 1896 that the first reference to the general solution

\( \hspace{11cm} ax^2 + bx + c = 0 \) \( \hspace{4cm}(a, b, c \hspace{0.5cm} are \hspace{0.5cm} constants, a \neq 0)\)

as \( \hspace{11cm} x = \frac{-b \pm \sqrt{b^2 - 4ac} }{2a} \)

was published by Henry Heaton in The American Mathematical Monthly (see information at http://en.wikipedia.org/wiki/Quadratic_equation and http://www.jstor.org/stable/2971099?seq=1 )

Key points and principle

  • There are three methods of solving quadratic equations:


    • (i) quadratic factorisation, where we can write

    \( \hspace{6cm}ax^2 + bx + c = a ( x - p )( x - q ) \) when p and q when p and q are rational numbers

    \(\hspace{6cm}\) which has solution \(x = p\) or x = q

    (ii) formula \( \hspace{2cm} x = \frac{-b \pm \sqrt{b^2 - 4ac} }{2a} \)

    (iii) completing the square, where we write

    \( \hspace{6cm}ax^2 + bx + c = a \lgroup x + \frac{b}{2a}\rgroup^2 - \frac{b}{4a} + c \)

  • Note that if ( x - a )( x - b ) = 0, then either x - a = 0 , or x - b = 0, giving x = a or x = b.

  • quadratic equation have 2, 1 or 0 real root according to the value " \( b^2 - 4ac \)":

    • (i) if \( b^2 > 4ac \), there are 2 real distinct roots

    (ii) if \( b^2 = 4ac \), there is 1 (repeated) real root

    (iii) if \( b^2 < 4ac \), there are no real root

    Facts to remember
  • If \( \lgroup x - a \rgroup \) \(\lgroup x - b \rgroup = 0 \) then x = a or x = b
  • The formula for solving the quadratic equation

    • \( \hspace{4cm} ax^2 + bx + c = a \)
    is
    \( \hspace{4cm} x = \frac { -b \pm \sqrt{b^2 - 4ac} }{2a} \)
  • The number of real roots of a quadratic is
    • 2 if \( b^2 > 4ac \)
    1 if \( b^2 = 4ac \)
    0 if \( b^2 < 4ac \)
